### 1. 项目简介

**蓓蕾网络货运管理系统** 是一款功能完善的开源、可商用、全流程覆盖的网络货运平台，支持货主与承运人之间的信息发布、交易撮合、全程监控、金融支付、信用评价等核心功能。系统历经多次迭代，已服务近 40 万客户与司机，成熟度高、业务覆盖广。采用前后端分离架构，全端代码开源。后端基于 **SpringBoot** 构建，具备高内聚、低耦合的特性，支持分布式部署。前端覆盖PC、H5、小程序和APP，基于 **Vue** 和 **uni-app** 开发。

### 2. 系统核心功能

-   **信息发布与货源管理**: 支持货主快速发布货源信息，司机和物流企业可在线查找、匹配合适货源，实现高效撮合与资源共享。
-   **线上交易与电子运单**: 实现线上签约、下单、支付、开票等全流程电子化，支持电子运单生成与实时跟踪，提升运输业务透明度与效率。
-   **车辆轨迹监控与预警**: 通过GPS定位和物联网技术实时监控车辆运行轨迹，提供异常偏离、超速、长时间停留等智能预警，保障运输安全。
-   **财务对账与支付管理**: 系统自动生成结算账单，支持多种支付方式与对账功能，帮助企业实现资金流闭环管理，降低财务风险。
-   **信用评价与投诉处理**: 建立多维度信用体系，对货主、司机、承运商等进行综合评分；提供投诉举报渠道，保障交易公平与服务质量。
-   **多维度数据统计与日志分析**: 内置数据分析引擎，支持业务、运营、财务、用户行为等多维度统计与可视化展示，辅助管理决策。
-   **支持交通运输部、税务等数据调取**：对接交通运输部、税务等监管接口，支持电子运单、发票、从业人员、车辆资质等数据联动与合规上报。

### 6. 功能模块
#### 6.1管理端
1. 托运人管理：审核、账号管理、分配司机、上传合同  
2. 实际承运人管理：账号、银行卡、自然人审核  
3. 承运审核：司机信息审核、车辆关联、身份调整  
4. 驾驶员管理：司机信息上传、车辆管理  
5. 车辆管理：车辆信息审核、入网查询  
6. 车辆服务：轨迹查询、位置查询、运力池  
7. 订单管理：支持指派、抢单、竞价，运单轨迹监控、合同、保险  
8. 财务管理：对账、月账单、应收应付、线上线下结算  
9. 数据调取：电子运单、资金流水、位置信息、车辆、驾驶员  
10. 日志统计：操作日志、登录日志、异常日志  
11. 咨询评价：信用评价、投诉、留言  

#### 6.2货主端 APP
- 首页：查看货源单  
- 发布：指派、抢单、竞价模式  
- 消息：系统提醒  
- 我的：个人信息、待结算、地址、部门、对账单  

#### 6.3司机端 APP
- 运单：接单、运输中、已结算
- 货源大厅：平台和全网货源
- 消息：接单、提货、交易提醒
- 我的：身份认证、银行卡、车辆、合同、反馈

### 8. 部署说明

#### 8.1环境准备与部署

在开始部署前，请确保您的服务器或本地环境已具备以下基础条件：

| 环境 | 推荐版本 | 说明 |
|------|-----------|------|
| 操作系统 | CentOS 7+ / Ubuntu 20.04+ / Windows 10+ | 支持常规 Java 与 Node.js 运行环境 |
| JDK | 1.8 或以上 | 需配置 `JAVA_HOME` 环境变量 |
| Node.js | 16+ | 前端项目构建必需 |
| MySQL | 5.7 / 8.0 | 数据持久化 |
| Redis | 5.0+ | 缓存与消息队列支持 |


详细的本地部署指南，请参考官方文档：  
👉 [**部署文档 → 环境准备**]

---

#### 8.2数据库初始化

- **推荐方式**：使用项目提供的初始化 SQL 脚本，导入至本地或远程 MySQL 实例中。  
- **脚本位置**：  
  👉 [**数据库脚本（Gitee）**]

**执行示例：**
```bash
mysql -u root -p < init.sql
```
#### 8.3后端部署

##### 8.3.1 进入项目目录
```bash
cd beilei-server
```
##### 8.3.2 修改配置文件

打开 src/main/resources/application.yml，配置数据库与 Redis 信息：
```bash
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/beilei?useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: yourpassword
  redis:
    host: localhost
    port: 6379
```
##### 8.3.3 打包并启动服务
```bash
mvn clean package -DskipTests
java -jar target/beilei.jar
```
##### 8.3.4 验证启动成功

访问接口：

http://localhost:8080

#### 8.4前端部署

#### 8.4.1进入前端项目目录
```bash
cd beilei-ui
```
#### 8.4.2安装依赖
```bash
npm install
```
#### 8.4.3配置后端接口地址

修改 .env.development 或 .env.production 文件中的后端 API 地址：
```bash
VITE_GLOB_API_URL=http://localhost:8080
npm install
```
#### 8.4.4本地启动项目
```bash
npm run dev
```
默认访问地址：

http://localhost:3100

#### 8.4.5构建生产环境
```bash
npm run build
```
打包后的静态文件位于 dist/ 目录，可通过 Nginx、Tomcat 或其他 Web 服务部署。
