## 一、系统简介
蓓蕾网络货运系统是一款基于互联网技术的物流管理平台，支持货主与承运人之间的信息发布、交易撮合、全程监控、金融支付、信用评价等核心功能。  

系统历经多次迭代，已服务近 **40 万客户与司机**，成熟度高、业务覆盖广。  

### 系统核心功能
- 信息发布与货源管理  
- 线上交易与电子运单  
- 车辆轨迹监控与预警  
- 财务对账与支付管理  
- 信用评价与投诉处理  
- 多维度数据统计与日志分析  
- 支持交通运输部、税务等数据调取  
- 可视化大屏展示

## 四、功能模块

### （一）管理端
1. 托运人管理：审核、账号管理、分配司机、上传合同  
2. 实际承运人管理：账号、银行卡、自然人审核  
3. 承运审核：司机信息审核、车辆关联、身份调整  
4. 驾驶员管理：司机信息上传、车辆管理  
5. 车辆管理：车辆信息审核、入网查询  
6. 车辆服务：轨迹查询、位置查询、运力池  
7. 订单管理：支持指派、抢单、竞价，运单轨迹监控、合同、保险  
8. 财务管理：对账、月账单、应收应付、线上线下结算  
9. 数据调取：电子运单、资金流水、位置信息、车辆、驾驶员  
10. 日志统计：操作日志、登录日志、异常日志  
11. 咨询评价：信用评价、投诉、留言  

### （二）货主端 APP
- 首页：查看货源单  
- 发布：指派、抢单、竞价模式  
- 消息：系统提醒  
- 我的：个人信息、待结算、地址、部门、对账单  

### （三）司机端 APP
- 运单：接单、运输中、已结算  
- 货源大厅：平台和全网货源  
- 消息：接单、提货、交易提醒  
- 我的：身份认证、银行卡、车辆、合同、反馈
